Windows上将linux目录映射网络驱动器

发表于2021-08-27|更新于2022-03-08|Linux的基本使用
|字数总计:141|阅读时长:1分钟|阅读量:

准备工作

安装samba

samba介绍

sudo apt-get install samba

修改配置文件

 sudo vim /etc/samba/smb.conf
# 在文件末尾添加以下内容
[raspberrypi]
    path = /home/tangshp
    public = yes
    writeable = yes
    browseable = yes
    guest ok = yes
    create mask = 0777
    directory mask = 0777
    available = yes
    comment = share folder
    valid user = tangshp

public的命名后续会被用到

添加samba用户

sudo smbpasswd -a tangshp

重启smbd生效

sudo /etc/init.d/smbd restart

在windows上映射网络驱动器

注意ip地址后的路径必须使用之前的public，必须要一致;要勾选使用其它凭据登入
