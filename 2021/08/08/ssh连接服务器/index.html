ssh连接服务器

发表于2021-08-08|更新于2021-09-11|Linux的基本使用
|字数总计:330|阅读时长:1分钟|阅读量:

1.准备工作

1.1.查看是否安装openssh-server

ls /etc/ssh/
moduli      sshd_config         ssh_host_ecdsa_key.pub  ssh_host_ed25519_key.pub  ssh_host_rsa_key.pub
ssh_config  ssh_host_ecdsa_key  ssh_host_ed25519_key    ssh_host_rsa_key          ssh_import_id

查看是否有sshd_config，没有则按照如下安装openssh-server

sudo apt-get install openssh-server

1.2.查看公钥记录文件路径

cat /etc/ssh/sshd_config
...
#AuthorizedKeysFile     .ssh/authorized_keys
...

| 公钥被存放在 .ssh/authorized_keys

ssh-keygen -t rsa -C "your_email@example.com" # 其它命令说明

代码参数含义：
-t 指定密钥类型，默认是 rsa ，可以省略。
-C 设置注释文字，比如邮箱。
