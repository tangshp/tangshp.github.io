时间

获取系统时间

  • gettimeofday
    函数获取至1970年至今的时间, 并将其保存在tv中; 函数原型为int gettimeofday(struct timeval *tv, struct timezone *tz);,头文件位于#include <sys/time.h>, 其中对应的两个结构体如下:
     struct timeval {
   time_t      tv_sec;     /* seconds */
   suseconds_t tv_usec;    /* microseconds */
};
struct timezone {
   int tz_minuteswest;     /* minutes west of Greenwich */
   int tz_dsttime;         /* type of DST correction */
};
    • 参数tz是一个历史产物, 在早期的UNIX系统实现中使用其来获取系统的时区信息, 目前已被废弃, 因此在调用时始终置为NULL;
    • return 0 for success, or -1 for failure

时间戳和年月日时分秒的转换

  • 时间戳转化为年月日时分秒使用struct tm *localtime(const time_t *timep);, 头文件位于#include <time.h>, 结构体tm定义如下:
    struct tm {
   int tm_sec;    /* Seconds (0-60) */
   int tm_min;    /* Minutes (0-59) */
   int tm_hour;   /* Hours (0-23) */
   int tm_mday;   /* Day of the month (1-31) */
   int tm_mon;    /* Month (0-11) */
   int tm_year;   /* Year - 1900 */
   int tm_wday;   /* Day of the week (0-6, Sunday = 0) */
   int tm_yday;   /* Day in the year (0-365, 1 Jan = 0) */
   int tm_isdst;  /* Daylight saving time */
};

    注意以上时间的区间

man localtime
tm_sec    The number of seconds after the minute, normally in the range 0 to 59, but can be up to 60 to allow for leap seconds.

tm_min    The number of minutes after the hour, in the range 0 to 59.

tm_hour   The number of hours past midnight, in the range 0 to 23.

tm_mday   The day of the month, in the range 1 to 31.

tm_mon    The number of months since January, in the range 0 to 11.

tm_year   The number of years since 1900.

tm_wday   The number of days since Sunday, in the range 0 to 6.

tm_yday   The number of days since January 1, in the range 0 to 365.

tm_isdst  A  flag that indicates whether daylight saving time is in effect at the time described.  The value is positive if daylight saving time is in ef‐
         fect, zero if it is not, and negative if the information is not available.
  • 年月日时分秒转化为时间戳使用time_t mktime(struct tm *tm);, 头文件位于#include <time.h>, 结构体time_t实际上是long int

demo

#include <time.h>
#include <sys/time.h>
#include <stdio.h>

int main(int argc, char **argv)
{
    struct timeval tv;
    struct timezone zone;
    gettimeofday(&tv, &zone);
    printf("tz_minuteswest:%d tz_dsttime:%d\n", zone.tz_minuteswest, zone.tz_dsttime);

    struct tm *ptLt = localtime(&(tv.tv_sec));

    printf("%04d-%02d-%02d %02d:%02d:%02d\n", ptLt->tm_year + 1900, ptLt->tm_mon + 1, ptLt->tm_mday,
           ptLt->tm_hour, ptLt->tm_min, ptLt->tm_sec);

    printf("wday:%d yday:%d\n", ptLt->tm_wday, ptLt->tm_yday);
    if (1 == ptLt->tm_isdst)
        printf("is dst.\n");
    else if (0 == ptLt->tm_isdst)
        printf("is not dst.\n");
    else
        printf("the information is not available.\n");

    time_t stamp = mktime(ptLt);

    printf("tv_sec:%ld\n", tv.tv_sec);
    printf("stamp :%ld\n", stamp);

    return 0;
}
