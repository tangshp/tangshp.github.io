strcpy和strdup

strcpy和strdup都是拷贝字符串的内容(duplicate a string)
1.函数声明对比

#include <string.h>
char *strdup(const char *s);
----------------------------
#include <string.h>
char *strcpy(char *dest, const char *src);

2.使用对比
使用strcpy之前必须为dest分配足够的内存;而strdup内部实现了内存的分配

3.示例

char src[] = "compare strdup to strcpy.";
size_t len = sizeof(src);

char *dest_cpy = NULL;
dest_cpy = (char *)malloc(len); // 内存分配
strcpy(dest_cpy, src);
printf("dest_src:[%s]\n", dest_cpy);

char *dest_dup = NULL;
dest_dup = strdup(src);
printf("dest_dup:[%s]\n", dest_dup);

malloc, alloc, remalloc, free

函数原型

这四个函数在stdlib.h中被声明

#include <stdlib.h>

void *malloc(size_t size);
void free(void *ptr);
void *calloc(size_t nmemb, size_t size);
void *realloc(void *ptr, size_t size);

区别

  • 1.malloc分配指定字节数size的存储区, 此存储区中的初始值不确定;malloc分配的是一块连续的内存,malloc(size)分配的内存可能比实际的size要多一点(这取决于编译器,不要指望一定会多一点);如果内存池中没能提供指定数量的内存,则返回NULL(确定malloc的返回值是否为NULL非常重要);
  • 2.alloc向内存池申请nmemb个大小为size的内存区域, 并给内存区域填充0
  • 3.realloc被用作重新分配内存;
    • 3.1.原来的内存块可以扩大缩小, 则原地址ptr和返回地址值相等; 如果是缩小一块内存块，则将该内存块的后半部分直接拿掉，剩余部分内存块及其内容保留; 如果是扩大一个内存块，则将原来的内存块保留,原内存原先的内容保留,在他的后边新增一块内存块（但是新增的内存块并未初始化）
    • 3.2.原来的内存块无法扩大缩小, realloc会重新开辟一个新的内存空间，并把原来的内存空间的内容拷贝到新的内存空间里(在调用完realloc后就不能使用指向就内存的指针ptr，而是用返回的新的指针。)
    • 3.3.realloc在内存缩小时,内部实现了对其尾部内存的释放工作;
    • 3.4.如果realloc的第二个参数为NULL,则realloc等于malloc

给free传入一个NULL参数不会产生任何效果

int *pi;
pi = (int *)malloc(100);
if (NULL == pi)
{
	printf("Out of memory.\n");// 内存不足 = Out of space
	exit(1);
}

常见错误

  • 对NULL进行解引用操作
  • 对分配的内存进行操作时越过边界:例如,pi = (int *)malloc(25*sizeof(int)),进行下标引用操作时,下标小于0或则大于24将引起两种类型的问题;
    • 第一种:被访问的内存可能保存了其它变量的值,对它进行修改将破坏那个变量的值,这种bug很难发现
    • 第二种:在malloc和free的有些实现中,使用链表来维护可用的内存池,对分配之外的内存区域进行访问可能破坏这链表,这可能产生异常而导致退出;
  • 释放并非动态内存分配的内存,传递给free的值必须是malloc、alloc、realloc返回的值
  • 释放动态内存分配的内存的一部分:例如,
    int *pi = (int *)malloc(25*sizeof(int));
free(pi+5);
  • 一块动态内存被释放后被继续使用,可能有多个指针指向同一片由动态分配的内存区域,这些指针散布在程序的各处,确保对其中任一个指针进行free操作前,停止对其余指针的使用;
  • 内存泄露(memory leak):由动态内存分配的内存区域,在不需要使用时,应该及时释放;
    • 在那些执行程序共用一片内存区域的操作系统,内存泄露将一点点地榨干可用内存,此时只能重启操作系统才能释放内存;
    • 在那些能够记住每个可执行程序的内存段的操作系统,当程序中止时,将回收分配给该可执行程序的内存;尽管如此,内存被耗尽时,该可执行程序无法执行下去,已完成的工作将统统丢失

编码小技巧

  • 设置free后的指针为NULL
    被free后的指针,只是放弃了指针对这个内存的占用, 放弃之后,内存的值会改写成随机值。但是指针本身并没有被删除,指针仍然指向原来的那块内存;例如:
    #include <stdio.h>
#include <stdlib.h>
int main(int argc, char **argv)
{
    int *pi = (int *)malloc(sizeof(int));
    *pi = 100;
    printf("address: %p, value: %3d.\n", pi, *pi);
    free(pi);
    printf("address: %p, value: %d.\n", pi, *pi);
}
===
address: 0xaaab1e1eb2a0, value: 100.
address: 0xaaab1e1eb2a0, value: 0.

    如果指针没有被设置为NULL,如果同时不小心再次被free,则会出现double free的core dumped;如下:

int *pi = (int *)malloc(sizeof(int));
*pi = 100;
printf("address: %p, value: %3d.\n", pi, *pi);
free(pi);
printf("address: %p, value: %d.\n", pi, *pi);
free(pi);
===
address: 0xaaaada6ac2a0, value: 100.
address: 0xaaaada6ac2a0, value: 0.
free(): double free detected in tcache 2
[1]    20014 abort (core dumped)  ./draft

参数说明

  • 函数/参数说明
    • va_list list用作定位函数的参数
    • va_start(list, format);初始化list,同时让list指向当前变参数的第一个参数(即”this”)
    • va_arg(list, char *);找到了第一个参数是char *类型,取出第一个参数,并指向下一个变参数;
    • va_end(list);释放指针类型list(等同于ap = (va_list)0)
#include <stdio.h>
#include <stdarg.h>
#include <string.h>

void var_test(char *format, ...)
{
    va_list list;
    va_start(list, format);
    char *ch;
    printf("[%s]\n", format);
    while (1)
    {
        ch = va_arg(list, char *);
        if (strcmp(ch, "") == 0)
        {
            break;
        }
        printf("[%s]\n", ch);
    }
    va_end(list);
}

int main()
{
    var_test("test", "this", "is", "a", "test", "");
    return 0;
}

变参数示例

  • printf实现和自定义日志
    #include <stdio.h>
#include <stdarg.h>
#include <string.h>

enum Log_Level{
    I,
    E,
    W
};

int Log(int log_level, ...){
    int result = 0;
    switch (log_level){
    case I:
        printf("[ \033[1m\033[32mI\033[0m ] : ");
        break;
    case E:
        printf("[ \033[1m\033[31mE\033[0m ] : ");
        break;
    case W:
        printf("[ \033[1m\033[33mW\033[0m ] : ");
        break;
    default:
        break;
    }
    va_list args;
    va_start(args, log_level); // 初始化 args
    char *format;
    format = va_arg(args, char *);// 获取变参数的第一个参数
    vprintf(format, args);
    va_end(args);// 置空 args
    return 0;
}

int print(char *format, ...){
    int result = 0;
    va_list args;
    va_start(args, format);
    result = vprintf(format, args);
    va_end(args);
}

int main(){
    print("%-s :%-s\n", "level", "       content");
    Log(I, "%-5s, hello world\n", "Info");
    Log(E, "%-5s, hello world\n", "Error");
    Log(W, "%-5s, hello world\n", "Warn");
    return 0;
}
