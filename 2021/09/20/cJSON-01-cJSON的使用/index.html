cJSON-01-cJSON的使用

发表于2021-09-20|更新于2021-09-21|cJSON
|字数总计:684|阅读时长:3分钟|阅读量:

cJSON是一个轻量级的使用c语言编写的JSON解析器;
本系列笔记以cJSON开源项目为JSON解析库; JSON数据格式参考JSON

cJSON结构体

typedef struct cJSON
{
    struct cJSON *next;
    struct cJSON *prev;

    struct cJSON *child;

    int type;

    char *valuestring;
    int valueint;
    double valuedouble;

    char *string;
} cJSON;

视频数据类型

以如下视频数据类型为示例讲解一个简单的cJSON的使用与部分函数的实现过程

{
        "name": "Jack (\"Bee\") Nimble",
        "format":       {
                "type": "rect",
                "width":        1920,
                "height":       1080,
                "interlace":    false,
                "frame rate":   24
        }
}

cJSON_CreateObject

cJSON_CreateObject创建了一个json对象root
1.函数cJSON_New_Item给cJSON类型分配内存, 并初始化为0

CJSON_PUBLIC(cJSON *)
cJSON_CreateObject(void)
{
    cJSON *item = cJSON_New_Item(&global_hooks);
    if (item){
        item->type = cJSON_Object;
    }
    return item;
}

value

"name": "Jack (\"Bee\") Nimble"是一个cJSON数据类型, 使用cJSON_CreateString("Jack (\"Bee\") Nimble")创建一个cJSON类型的数据;

CJSON_PUBLIC(cJSON *)
cJSON_CreateString(const char *string){
    cJSON *item = cJSON_New_Item(&global_hooks);
    if (item){
        item->type = cJSON_String;
        item->valuestring = (char *)cJSON_strdup((const unsigned char *)string, &global_hooks);
        if (!item->valuestring){
            cJSON_Delete(item);
            return NULL;
        }
    }
    return item;
}

1.函数cJSON_New_Item给cJSON类型分配内存, 并初始化为0
2.指定type为cJSON_String
3.函数cJSON_strdup为指针valuestring分配内存, 并把string的字符拷贝给valuestring
4.函数返回cJSON类型的指针item

此时item的valuestring和type已被初始化, 但键key[string]还没有初始化;即仅仅是"Jack (\"Bee\") Nimble"

绑定到root

cJSON_CreateString("Jack (\"Bee\") Nimble")创建了一对key-value(还没有设置key), 该cJSON*以child的身份存在于root中;

  • 调用add_item_to_array之前, 为item设定key

    cJSON_AddItemToObject(root, "name", cJSON_CreateString("Jack (\"Bee\") Nimble"));

CJSON_PUBLIC(cJSON_bool)
cJSON_AddItemToObject(cJSON *object, const char *string, cJSON *item)
{
    return add_item_to_object(object, string, item, &global_hooks, false);
}

static cJSON_bool add_item_to_object(cJSON *const object, const char *const string, cJSON *const item, \
    const internal_hooks *const hooks, const cJSON_bool constant_key){
    char *new_key = NULL;
    int new_type = cJSON_Invalid;
    // 为key分配内存
    new_key = (char *)cJSON_strdup((const unsigned char *)string, hooks);

    new_type = item->type & ~cJSON_StringIsConst;

    if (!(item->type & cJSON_StringIsConst) && (item->string != NULL)){
        hooks->deallocate(item->string);
    }

    item->string = new_key;
    item->type = new_type;
    return add_item_to_array(object, item);
}

  • add_item_to_array
    ** child为空
    当child为空时(第一次调用), 如图-add_item_to_array_first所示, 调用前灰色部分为空; 调用后非空

    static cJSON_bool add_item_to_array(cJSON *array, cJSON *item)
{
    cJSON *child = NULL;
    child = array->child;

    if (child == NULL){
        /* list is empty, start new one */
        array->child = item;
        item->prev = item;
        item->next = NULL;
    }else{
        /* append to the end */
        if (child->prev){
            suffix_object(child->prev, item);
            array->child->prev = item;
        }
    }
    return true;
}


    ** child非空
    当child为非空时(非第一次调用), 如图add_item_to_array_no_first所示, 使用array的第一个item子节点的prev指针来定位最后一个item子节点(如红色指向所示), 形成双向链表

    static void suffix_object(cJSON *prev, cJSON *item){
    prev->next = item;
    item->prev = prev;
}
