matplotlib

Times New Roman字体问题

  • 使用Times New Roman时, 报找不到 [‘Times New Roman’]的错误
    plt.ylabel('Clocks($\mu S$)', fontdict={'family' : 'Times New Roman', 'size'   : 16})
    findfont: Font family ['Times New Roman'] not found. Falling back to DejaVu Sans.
  • 解决方法
    • 把windows中C:\Windows\Fonts的Times New Roman字体拷贝到以下路径中(使用pip3 show matplotlib查看matplotlib的安装路径)
      ls ./time*
./timesbd.ttf  ./timesbi.ttf  ./timesi.ttf  ./times.ttf
sudo cp ./times* /usr/local/lib/python3.8/dist-packages/matplotlib/mpl-data/fonts/ttf
    • 删除matplotlib的缓存:
      rm -rf ~/.cache/matplotlib
