tldr

npm install -g tldr

node

# 下载nodejs
➜ Tools wget https://nodejs.org/dist/latest-v12.x/node-v12.22.6-linux-arm64.tar.gz

# 解压
➜ Tools tar zxvf ./node-v12.22.6-linux-arm64.tar.gz

# 重命名
➜ Tools mv node-v12.22.6-linux-arm64 node

# 移动到/usr/local
➜ Tools sudo mv ./node /usr/local

# 添加环境变量
➜ Tools cat ~/.zshrc | tail -n 1
PATH=$PATH:/usr/local/node/bin
# 查看node版本

➜ Tools node -v
v12.22.6

pm2

  • PM2是node进程管理工具，可以利用它来简化很多node应用管理的繁琐任务，如性能监控、自动重启、负载均衡等，而且使用非常简单;如何使用参考Linux-命令;
    npm install -g pm2
  • 安装完成后,会自动创建下面目录
    ➜ .pm2 pwd
/home/pi/.pm2
➜ .pm2 ls
