pointer-like classes

智能指针

加入有一个shared_ptr对象sp,为了满足使用者以下使用习惯:

  • *sp
  • sp->method()

C++为了支持以上使用习惯, 通过以下函数应对

  • T &operator*() const
  • T *operator->() const


xxx_like_class.h

xxx_like_class_main.cpp

  • learn::shared_ptr<learn::Foo> sp(new learn::Foo(10));
    • 首先调用Foo的构造函数,new learn::Foo(10)获取到一个Foo指针
    • 调用shared_ptr的构造函数
  • learn::Foo foo(*sp);
    • 编译器会把*sp转化为*px的引用
    • 以*px的引用为实参, 调用拷贝构造函数
  • sp->method();
    • 编译器会把sp->转化为px
    • 符号->会继续作用, 所以结果为px->method
