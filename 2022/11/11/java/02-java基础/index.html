类型转化

  • 自动类型转换:转化时不需要加类型转化(var1 = var2)

    • 容量小的类型自动转化成容量大的数据类型
    • byte、short、int、char、long是精确类型, float和long是近似类型,精确类型转化为近似类型可能会有精度丢失
      i = 314159265;
float f = i;
System.out.printf("f:%E\n", f);
// 运行
f:3.141593E+08
// f = 314159300
// i = 314159265
    • byte、short、int三者在计算时,都会被转化为int类型,所以表达式的结果也是int类型,赋值时需注意,否则报错
      byte b = 127;
short s = b;
int i = s;

s = b + s + i;
// java: 不兼容的类型: 从int转换到short可能会有损失

  • 强制类型转换(var1 = (type)var2)

    byte b = 127;
short s = b;
int i = s;

s = (short)(b + s + i);
// java: 不兼容的类型: 从int转换到short可能会有损失

    强制类型转化时有可能导致精度降低和数据溢出

^ 异或
! 非
&& 短路与 左侧满足后, 右侧的不会计算

位运算符

位运算符 说明
~ 按位取反
& 按位与
| 按位或
^ 按位异或
<< 左移运算,低位补0
>> 右移运算,左侧空出的补符号位
>>> 无符号右移运算,左侧空出的0

  • 取反

    int integer = -1;
System.out.printf("integer:0X%X\n",integer);
System.out.printf("integer:0X%X\n",~integer);
integer = Integer.MIN_VALUE;

integer = Integer.MAX_VALUE;
System.out.printf("integer:0X%X\n",integer);

    int integer1 = 0X7FFFFFFF;

方法的定义

import java.util.Scanner;

public class Main {
    public static void main(String[] args) {
        System.out.printf("sqrt(%f):%f",5f, SqrtByBisection(5f));
    }


    final static Float eps = 0.001f;
    public static Float SqrtByBisection(Float n) //用二分法
    {
        if(n<0) //小于0的按照你需要的处理
            return n;
        Float mid,last;
        Float low = 0f,up = n;
        mid=(low+up)/2;
        do
        {
            if(mid*mid>n)
                up=mid;
            else
                low=mid;
            last=mid;
            mid=(up+low)/2;
        }while(Math.abs(mid-last) > eps);//精度控制
        return mid;
    }
}
