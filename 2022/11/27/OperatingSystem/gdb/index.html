基本命令

调试代码

#include <stdio.h>

void swap(int *pA, int *pB)
{
        int temp = *pA;
        *pA = *pB;
        *pB = temp;
}


int main(int argc, char **argv)
{
        printf("Hello World!\n");
        int arr[4] = { 1, 2, 3, 4};
        for(int i = 0; i < 4; i++)
        {
                printf("%d\n", arr[i]);
        }
        swap(&arr[0], &arr[3]);
        return 0;
}

生成调试信息

gcc -g ./main.c -o main

进入调试模式

  • gdb --args executable arg1 arg2 进入调试模式
  • run指令会一直执行程序, 直到执行结束。 如果程序中手动设置有断点, 则run指令会执行程序至第一个断点处；
  • start 指令会执行程序至main()主函数的起始位置，即在main()函数的第一行语句处停止执行（该行代码尚未执行）。
  • print 指令可以查看变量状态
    ➜  P2 gdb --args ./main firstArg secondArg

配置

参考ubuntu20.04 如何生成core文件

  • 配置core文件的限制
    • 临时:ulimit -c unlimited
    • 永久:/etc/profile文件中添加ulimit -c unlimited
  • 修改/etc/default/apport文件，将enabled 设置为0
